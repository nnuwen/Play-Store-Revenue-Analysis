
# Play-Store-Revenue-Analysis
Exploratory Analysis of Google Play Store revenue data using SQL and Python. The analysis uses SQL for data processing and python for data analysis and visualization. 

This project works through to identify patterns surrounding application revenue, user engagement, ratings, and screen time within Google Play Store's data. 

# Business Problem 
The goal of this project is to understand which factors or application characteristics contributed to revenue decrease or increase as well as customer engagement. 

## Tools Used 
- SQL
- Python
- Juptyer Notebook

## Project Files 
-`data_preprocessing.sql` – SQL queries used to inspect, clean, and prepare the data
- `Exploratory_Data_Analysis.ipynb` – Python notebook containing exploratory analysis and visualizations
- `playstore_revenue_analysis.csv` – Dataset used for the analysis

## Analysis Process 
1. Reviewed dataset and checked for errors, repeats, and duplicates
2. cleaned and prepared the data using SQL
3. analyzed the applications performances using python
4. compared relationships between different data sets within revenue, screen time, and other variables
5. created visualizations
